# ToDo

  <img src="https://github.com/svnomore/ToDo/blob/master/ToDo/icon.png?raw=true" alt="ToDo">

## Features
● Add and Remove your tasks to list

● Setting font size

● Deleted tasks list

● Reminder

● Run in background

● Startup

## How To Use
Download latest version from "Releases", run setup.exe, follow the instructions.
